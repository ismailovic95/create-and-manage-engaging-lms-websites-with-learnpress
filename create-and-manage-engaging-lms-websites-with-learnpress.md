# Create and Manage Engaging LMS Websites with LearnPress

> This course is a detailed and easy tutorial to get you all setup and going with the use of LearnPress LMS Plugin.

#### COURSE DESCRIPTION
A series of Videos from ThimPress, give you a detailed tutorial to create an LMS Website with LearnPress &#8211; LMS &amp; Education WordPress Plugin.

This course is a detailed and easy tutorial to get you all setup and going with the use of [**LearnPress LMS Plugin**](https://wordpress.org/plugins/learnpress/). It is a free and simple plugin to help you create an **Online Courses Website** step by step. The tutorial guides you through the configuration of the plugin, creation of Courses, Lessons, Quizzes, and finally guides you on how to boost up your Website with [**Premium LearnPress Add-ons**](https://thimpress.com/wordpress/plugins/learnpress/?utm_source=Eduma&amp;utm_medium=Course&amp;utm_campaign=LMSWebsite) brought to you by ThimPress (creator of LearnPress). It also shows how you could configure additional items like the course layouts and featured images.
